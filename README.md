# 🌱 Agro Forte - Futuro Sustentável

Projeto educativo sobre irrigação sustentável utilizando captação e reutilização da água da chuva.

## Objetivo

Demonstrar como a agricultura pode aumentar a produção e ao mesmo tempo preservar o meio ambiente por meio do aproveitamento da água da chuva.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Simulador de captação de água da chuva
- Tanque interativo com animação
- Cálculo estimado do volume armazenado
- Informações sobre sustentabilidade
